# CutterAI
Specify the video path, enter a prompt with what to look for, and after a short while, a compiled highlight reel will appear in the output folder (it might be a bit hit-or-miss due to current AI model limitations).
You need to provide a Telegram bot token in the config (as the bot serves as your UI) and get an API key from [aistudio.google.com](https://aistudio.google.com).
Also download FFmpeg from [gyan.dev](https://www.gyan.dev/ffmpeg/builds/), extract `ffmpeg.exe` and `ffprobe.exe` from the `bin` folder, and drop them directly into the root directory of this project.
